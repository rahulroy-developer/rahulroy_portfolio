# 🚀 Modern Developer Portfolio

![Project Status](https://img.shields.io/badge/Status-Active-success)
![License](https://img.shields.io/badge/License-MIT-blue)
![Tech Stack](https://img.shields.io/badge/Tech-HTML5%20%7C%20CSS3%20%7C%20JS-yellow)

A high-performance, fully responsive personal portfolio website designed for software developers. Built with **Semantic HTML5**, **Modern CSS3** (Grid/Flexbox), and **Vanilla JavaScript**, featuring a "Glassmorphism" aesthetic, dark mode support, and a working contact form.

🔗 **Live Demo:** [https://rahul1-portfolio.netlify.app/](https://rahul1-portfolio.netlify.app/)

## ✨ Key Features

* **🎨 Modern UI/UX:**
    * **Glassmorphism:** Frosted glass effects on cards and navigation.
    * **Bento/Grid Layout:** A responsive grid system for displaying projects.
    * **Browser-Style Cards:** Projects are displayed inside stylish "mini-browser" windows.
* **🌓 Dark & Light Mode:** Fully integrated theme switcher with persistence (saves preference to LocalStorage).
* **📱 Fully Responsive:** Optimized for all devices, from large 4k monitors to small mobile screens.
* **⚡ Performance:** Zero heavy frameworks (no React/Angular/Vue required), ensuring lightning-fast load times.
* **🔄 Dynamic Interactions:**
    * "Load More" functionality for the portfolio section.
    * Smooth scrolling navigation.
    * Scroll-to-top button.
    * Accordion-style skills section.
* **📧 Working Contact Form:** Integrated with **EmailJS** to send emails directly from the frontend without a backend server.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Icons:** [Unicons](https://iconscout.com/unicons)
* **Fonts:** [Google Fonts](https://fonts.google.com/) (Inter & Poppins)
* **Email Service:** [EmailJS](https://www.emailjs.com/)
